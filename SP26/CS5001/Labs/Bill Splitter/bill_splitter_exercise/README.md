# Bill Splitter Exercise - Setup Instructions

## Academic Integrity Notice
This exercise is designed to test YOUR programming skills. AI assistance tools are disabled in this project to ensure you're learning the fundamental concepts.

## Setup Instructions

### Step 1: Download the Project
1. Download the entire project folder
2. Extract it to your preferred location
3. Open VS Code

### Step 2: Open the Project Correctly
**IMPORTANT:** You must open the entire folder, not just the Python file!

1. In VS Code, go to: **File → Open Folder**
2. Select the `bill_splitter_exercise` folder (the whole folder)
3. Click "Select Folder" or "Open"

### Step 3: Verify AI is Disabled
Once the folder is open, you should see:
- A red status bar at the bottom of VS Code
- No autocomplete suggestions when typing
- No Copilot icon (if you have Copilot installed)

If you still see suggestions:
1. Make sure you opened the FOLDER, not just the file
2. Reload VS Code: **View → Command Palette → "Reload Window"**
3. Check that `.vscode/settings.json` exists in your project

### Step 4: Work on the Exercise
1. Open `bill_splitter_exercise.py`
2. Read the instructions carefully
3. Complete the `split_bill()` function
4. Run the file to test your solution: `python bill_splitter_exercise.py`
5. All tests should pass when your solution is correct
